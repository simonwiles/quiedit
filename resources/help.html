About

QuiEdit is a distraction-free, fullscreen text editor. It is created by Sebastiaan Mathôt <http://www.cogsci.nl/>.

Usage

QuiEdit is controlled using keyboard shortcuts, which are defined in the file keybindings.conf (see below).
